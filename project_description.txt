PROJECT: SonicKraft
DEVELOPER: S.K. (Srikar Kuchi)
TECH STACK: Python (Flask), JavaScript (ES6+), HTML5, CSS3 (Vanilla)
API INTEGRATIONS: Song.link (Odesli) for Cross-Platform Translation, Spotdl for Audio Streaming/Extraction

OVERVIEW:
SonicKraft is a high-performance web application designed for music enthusiasts who want total control over their library. Built with a focus on aesthetics and efficiency, SonicKraft provides a premium, unified interface for downloading Spotify content and translating music links across major platforms like Apple Music, YouTube Music, and SoundCloud.

KEY FEATURES:
1. Pro-Grade Downloads: Supports multiple audio formats including high-quality MP3 (320kbps), lossless FLAC, and M4A.
2. Cross-Platform Translator: Instantly converts Spotify links into equivalent links for YouTube, Apple Music, and SoundCloud using the Song.link API.
3. Real-Time Status Engine: Features a live equalizer and asynchronous polling to provide second-by-second updates on the download process.
4. Premium Glassmorphism UI: A state-of-the-art interface featuring dynamic aurora backgrounds, floating music note particles, and smooth CSS transitions.
5. Smart Library Management: Includes a local persistent storage system to track download history and lifetime statistics.
6. Intuitive UX: Supports Drag-and-Drop link processing, global keyboard shortcuts, and a fully responsive layout.

TECHNICAL CHALLENGES SOLVED:
- Bypassed strict platform rate limits by implementing decentralized audio routing through Piped instances.
- Developed a non-blocking background task system in Flask to handle massive 500+ song playlists without browser timeouts.
- Designed a custom CSS animation engine to maintain high performance while rendering complex visual effects like aurora orbs and particle systems.

PORTFOLIO SUMMARY:
SonicKraft demonstrates the intersection of robust backend logic and high-end frontend design. It showcases my ability to handle asynchronous processes, API integration, and modern UI/UX principles to build a tool that is both functionally powerful and visually stunning.
