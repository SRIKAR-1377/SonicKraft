Spotify Playlist Downloader - How It Works
==========================================

This project is a web application that allows users to seamlessly download a full Spotify playlist into a ZIP file. It consists of a modern Glassmorphism frontend UI and a Python backend.

1. The Technology Stack
-----------------------
- Frontend: HTML5, CSS3, and Vanilla JavaScript.
- Backend: Python 3 with the Flask framework.
- Core Download Engine: `spotdl` (A Python library that matches Spotify metadata to YouTube audio and downloads it).

2. How the User Interface Works (Frontend)
------------------------------------------
- The UI is designed with a "Glassmorphism" aesthetic (translucent elements, blur effects, and vibrant gradients on a dark background) to feel modern and premium.
- In `static/index.html`, we have an input field where you paste a Spotify playlist link, and a "Download" button.
- In `static/script.js`, when you click "Download", the JavaScript grabs your Spotify link and sends a background `POST` request (using the Fetch API) to our backend server at `/api/download`.
- While waiting for the server, the button changes to a loading spinner, and an animated progress bar appears so the user knows something is happening.

3. How the Server Works (Backend)
---------------------------------
- The `app.py` script starts a Flask web server, which listens for incoming requests on port 5000.
- When it receives the Spotify URL at the `/api/download` endpoint, it creates a unique, temporary folder for your session.
- It then executes the `spotdl` engine directly using Python (`import sys; sys.executable -m spotdl`). `spotdl` matches the songs to YouTube Music, downloads them in high quality, and applies proper ID3 tags (artist, album, artwork).
- Once `spotdl` finishes downloading all the tracks into the temporary folder, the server packages that folder into a neatly compressed `.zip` archive.
- Finally, the server cleans up the unzipped files to save space and replies to the frontend with a direct download link to the ZIP file.

4. Bringing It Together
-----------------------
- When the frontend gets the successful reply, it hides the loading animations and presents a "Save ZIP File" button.
- Clicking that button triggers the download, saving all your songs directly to your device!
